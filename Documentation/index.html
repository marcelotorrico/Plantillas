Basic Folder Structure

  • audio - Demo audio file, used in blog
  • css - All css files
  • font-face - Font directory
  • fonts - Font icons
  • images - all imagesjs - All javascript files
  • plugins -  3rd party plugins
  • sass - saas sources (compiled to css/style.css)

Plugins

  • jquery-ui
  • responsiveslides - custom built responsive slider. Used on home pages

SASS files

All SASS files are located under sass directory

  • bootstrap - bootstrap framework
  • vendor - all vendor styles
  • pages - specific styles related to seperate pages
  • components - all template components
  • _bootstrap.scss - calling bootstrap library files
  • _style.scss - main file
  • _settings.scss - settings file (you can change all theme settings here - fonts, colors, grid width and so on)

JS files

You can find all JS files under js and plugins directories

  • bootstrap.min.js - Minified bootstrap script
  • fastclick.js - script which optimize touch for menu on small devices
  • jquery-accessibleMegaMenu.js - script for megamenu (This project is maintained by adobe-accessibility)
  • jquery-migrate-1.2.1.min.js  - support migration
  • jquery-ui-1.10.4.custom.min.js - jquery custom mini
  • jquery.bxslider.min.js - box slider
  • jquery.elevatezoom.js - used on product page to zoom main product image
  • jquery.knob.js - used on coming soon page
  • jquery.ui.touch-punch.min.js - used on category page only
  • jquery.validationEngine-en.js -  form validation localization
  • jquery.validationEngine.js -  form validation
  • mediaelement-and-player.min.js - for player and media elements
  • modernizr-2.6.2.min.js - for devices and browsers adaptation
  • plugins.js - mobile_menu plugin, custom select
  • review.js - used for review
  • scripts.js - all custom javascript code is here
  • sly.min.js - for scroll
  • timer.js - used on comming soon page
